Chapter 5: Higher Order Functions
A large program is a costly program, leads to confusion, and obscures bugs.
If you abstract what you want to do the browser has to work harder, but your code is much prettier.
Experienced programmers can find optimum abstractions for code, using vanilla 'for' loops for speed when needed.
However, if you are dealing with the human user timescale, it is often better to do the abstract, maintainable method first!
You could write out functions for array filter, map, and forEach, but it's already built-in to the modern browser.
